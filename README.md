# Student Management Portal

## Project Overview

The Student Management Portal is a web-based application developed using HTML, CSS, and JavaScript. It helps administrators efficiently manage student records, attendance, marks, reports, settings, and contact information through a simple and user-friendly interface.

The application uses Local Storage to save data within the browser, eliminating the need for a backend database.

---

## Features

- Administrator Login
- Dashboard
- Student Registration
- Student List
- Student Profile
- Attendance Management
- Marks Management
- Reports
- Settings (Dark Mode / Light Mode)
- About Page
- Contact Form

---
